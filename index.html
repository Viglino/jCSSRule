jQuery.cssRule

A simple jQuery plugin to manipulate CSS css rules.
The plugin insert a new style sheet element on the page. It will be inserted where the js code is placed (head or body) and will override other CSS style sheets placed before.

  • Use $(selector).cssRule(property,value) or $(selector).cssRule({ property:value }) to add new rules for the selector.
  • $(selector).cssRule(property) to get the value for that key.
  • $(selector).cssRule(property,null) will remove the rule and use the default one defined in the css.
  • $("*").cssRule(null) will remove all previous rules.

Background

$("body").cssRule({ background:xxx });

Title

$("h1,h2").cssRule({ color:xxx, background:xxx });

Buttons

$("button").cssRule({ background:xxx });

Reset

$("*").cssRule(null);
